# UBCabExpress Driver Staff Connect

Static HTML site for Vercel.

Deploy:
1. Upload this folder to GitHub.
2. Import the repository in Vercel.
3. Framework Preset: Other.
4. Build Command: leave empty.
5. Output Directory: leave empty.

Realtime setup:
1. Create a Supabase project.
2. Open SQL Editor and run `supabase-schema.sql`.
3. Open `supabase-config.js`.
4. Replace `PASTE_SUPABASE_URL_HERE` and `PASTE_SUPABASE_ANON_KEY_HERE`.
5. Upload all files to GitHub again and redeploy on Vercel.
